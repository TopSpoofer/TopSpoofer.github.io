## 概述
现在基于关系的数据库都是以2维表的形式记录存储数据的，所以不能将树结构直接存入DBMS。本文主要介绍各种流行树形表的设计。

当前流行的树形数据库设计主要有以下几种：

```
1、Adjacency List
2、Path Enumeration
3、Nested Sets
4、Closure Table
```
